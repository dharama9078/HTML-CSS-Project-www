Information System.

The world wide web is an information system that enables content sharing over the Internet through user-friendly ways meant to appeal to users beyond IT specialists and hobbyists. It allows documents and oter web resorces to be accessed over the Internet according to specific rules of the HTTP.

The Web was invested by English computer scientist Tim_Berners-Lee while at CERN in 1989 and opened to the public in 1991. it was conceived as a universal linked informatin system Documents and other media content are made available to the network through web servers and can be accessed by program such as web browsers. Servers and resources on the World Wide Web are identified and located through characters string called URLs.

The original and the still very common document type is a web page formatted in HyperText Markup Language (HTML). This markup language supports plain text, image embedded video and audiocontents,and scripts that implement complex user interaction .The HTML language also supports hyperlinkswhich provides immediate access to other web resources. Web navigation,or web sufring,is the common practice of fallowing such hyperlinks across multiple websites.

History

The Web was invented by English copmuter scientist Tim Berners Lee while working at CERN. He was motivated by the problem of updating,and data files in tha large and constantly changing organization,as well as distributing them to collaborators outside CERN.In his design ,Berners-Lee dismissed the common tree structure approach ,used for instance in the existing CERNDOC documentation system and in the Unix filesystem, as well as approaches that relied in tagging files with keywords,as in the VAX/NOTES system.Insted he adopted concepts he had put into practice with his private ENQUIRE system(1980) built at CERN.When he became aware of Ted Nelson's hypertext model(1965),in which documents can be linked in unconstrained ways through hyperlink associated with hot spots embeded in the text,it helped to confirm the validity of his concept.
